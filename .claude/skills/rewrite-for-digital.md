# 改寫數位版內容

## 任務

將 `platform = 'docs'` 的週報原稿內容，改寫為適合網路閱讀的 `platform = 'digital'` 版本。

## 改寫原則

### 1. 慈濟風格（必須保留）

- **溫暖人文**：保持慈悲、感恩、善解、包容的語調
- **正向敘事**：強調付出與收穫、苦難中的希望
- **證嚴上人法語**：若原文引用上人開示，完整保留原文，不改寫
- **慈濟用語**：維持專有名詞（如「靜思語」「人間菩薩」「大愛」「慈濟人」）
- **謙和語氣**：避免誇大、煽情、商業化用語

### 2. GEO（Generative Engine Optimization）

針對 AI 生成式搜尋（如 Google SGE、Perplexity、ChatGPT Search）優化：

- **清晰的事實陳述**：確保 AI 能準確擷取關鍵資訊
- **結構化段落**：每段聚焦一個重點，方便 AI 摘要
- **明確的 5W1H**：Who/What/When/Where/Why/How 在前段完整呈現
- **可引用的句子**：重要觀點用完整句表達，便於 AI 引用
- **數據與事實**：具體數字、日期、地點清楚標示

### 3. AIO（AI Optimization）

讓 AI 系統更好理解內容：

- **語義清晰**：避免雙關語、隱晦表達
- **主題明確**：每篇文章有清楚的核心主題
- **實體標註**：人名、地名、組織名稱完整呈現
- **因果關係**：事件的起因、經過、結果邏輯清晰
- **上下文完整**：避免指代不明（如「他」「這件事」），必要時重複主詞

### 4. SEO（Search Engine Optimization）

傳統搜尋引擎優化：

- **標題優化**：
  - 包含關鍵字（地點、事件、組織）
  - 控制在 25-35 字
  - 具吸引力但不標題黨
- **首段摘要**：
  - 前 100 字涵蓋核心訊息
  - 自然融入主要關鍵字
- **內容結構**：
  - 使用小標題（H2/H3）分段
  - 段落適中（3-5 句）
  - 重點資訊靠前
- **關鍵字策略**：
  - 自然融入，不堆砌
  - 涵蓋：事件名稱、地點、人物、慈濟相關詞

## 改寫格式

### 標題改寫

原稿標題（報紙風格）：
```
慈濟「預鑄安心屋」完工 楠西山城震後新生
```

數位版標題（搜尋友善）：
```
台南楠西預鑄安心屋完工入住｜慈濟震災重建援助 6 戶居民迎新春
```

### 內容結構

```markdown
**核心摘要段**（前 100 字，回答 5W1H）

## 小標題一
內容段落...

## 小標題二
內容段落...

## 小標題三
內容段落...

---
關於慈濟基金會：[簡短介紹，固定文字]
```

## 輸出格式

```json
{
  "title": "改寫後的標題",
  "content": "改寫後的 markdown 內容"
}
```

## 不可改動

- 證嚴上人的原話/開示內容
- 直接引述的人物發言（引號內文字）
- 具體數字、日期、地名
- 慈濟專有名詞
- 圖片及其說明文字

## 範例

### 輸入（docs 版）

```
**臺南訊/**二○二五年一月廿一日清晨，強震重創嘉南山區，楠西、玉井與左鎮多處老宅倒塌，許多高齡居民一夕之間失去家園。為減少居民流離不安，慈濟與臺南市政府啟動「安居計畫」，同步推動原地重建與中繼安置，讓受災戶能盡早回歸生活軌道。

首度採預鑄工法克服山區挑戰

針對地形崎嶇、大型機具進入不易的山區，慈濟首度採用「預鑄工法」興建安心屋。
```

### 輸出（digital 版）

```markdown
2025 年 1 月 21 日強震重創台南山區，造成楠西、玉井、左鎮多處房屋倒塌。慈濟基金會與台南市政府合作啟動「安居計畫」，採用創新預鑄工法，於震後完成 6 戶安心屋重建，讓受災居民趕在農曆新年前入住新居。

## 預鑄工法首度應用於山區重建

面對山區地形崎嶇、大型機具難以進入的挑戰，慈濟首度採用「預鑄工法」——建材構件先在工廠製作完成，再運至現場快速組裝。這項工法大幅縮短工期，同時確保建築品質。
```

## 品質檢核

改寫完成後，確認：
- [ ] 保持慈濟溫暖人文的語調
- [ ] 5W1H 在首段完整呈現
- [ ] 標題包含關鍵字且具吸引力
- [ ] 使用小標題結構化內容
- [ ] 專有名詞和引述未被更動
- [ ] 內容長度適中（不過度精簡也不冗長）
