# Dashboard 後台設計

## 概述

週報管理後台介面設計，描述各頁面功能與元件需求。

## 頁面結構

```
/login              → 登入頁
/                   → 週報列表（首頁）
/weekly/:id         → 週報詳情（編輯文稿）
/weekly/:id/import  → 匯入狀態
/logs               → 審計日誌（可選）
```

## 1. 登入頁 `/login`

### 功能
- Google OAuth 登入按鈕
- 顯示系統名稱/Logo

### 元件
- 系統標題：「慈濟週報管理系統」
- Google 登入按鈕
- 錯誤訊息區（如：不在白名單內）

---

## 2. 週報列表 `/`（首頁）

### 功能
- 顯示所有週報期數
- 新增週報（匯入）
- 快速狀態切換

### 元件

#### 頂部操作列
- 「新增週報」按鈕 → 開啟匯入對話框
- 搜尋/篩選（依狀態）

#### 週報卡片列表
每張卡片顯示：
- 期數（如：第 117 期）
- 狀態標籤（draft / published / archived）
- 建立日期
- 文稿數量
- 操作按鈕：編輯、發布、封存、刪除

#### 新增週報對話框
- Google Doc URL 輸入欄
- 期數輸入欄（可自動偵測）
- 「開始匯入」按鈕
- 提交後導向匯入狀態頁

---

## 3. 週報詳情 `/weekly/:id`

### 功能
- 瀏覽/編輯該期所有文稿
- 切換原稿 (docs) / 數位版 (digital)
- 單篇重新 AI 改寫

### 元件

#### 頂部資訊列
- 期數標題
- 狀態標籤
- 「發布」/「封存」按鈕
- 「返回列表」連結

#### 分類 Tabs
- 8 個分類標籤頁
- 顯示各分類文稿數量

#### 版本切換
- Toggle：原稿 / 數位版
- 或並排顯示兩個版本

#### 文稿列表
每篇文稿顯示：
- 標題
- 內容預覽（摺疊/展開）
- 「編輯」按鈕
- 「重新改寫」按鈕（僅數位版）

#### 文稿編輯對話框
- 標題輸入欄
- 內容編輯器（Markdown）
- 預覽區
- 「儲存」/「取消」按鈕

---

## 4. 匯入狀態 `/weekly/:id/import`

### 功能
- 即時顯示匯入進度
- 完成後導向週報詳情

### 元件

#### 進度指示器
步驟列表（垂直或水平）：
1. ○ 下載 markdown
2. ○ 處理圖片
3. ● AI 解析中...（目前步驟高亮）
4. ○ 匯入原稿
5. ○ AI 改寫
6. ○ 匯入數位版
7. ○ 完成

#### 狀態訊息
- 目前步驟詳情（如：「正在改寫第 3/12 篇」）
- 錯誤訊息（如果失敗）

#### 操作按鈕
- 完成後：「查看週報」按鈕
- 失敗時：「重試」按鈕

---

## 5. 審計日誌 `/logs`（可選）

### 功能
- 檢視系統操作紀錄
- 篩選與搜尋

### 元件

#### 篩選列
- 時間範圍選擇
- 操作類型下拉選單
- 使用者篩選
- 週報期數篩選

#### 日誌表格
欄位：
- 時間
- 使用者
- 操作類型
- 說明
- 詳情按鈕

---

## 共用元件

### 導航列
- Logo / 系統名稱
- 目前使用者
- 登出按鈕

### 確認對話框
- 用於刪除、發布等重要操作
- 標題、說明文字、確認/取消按鈕

### Toast 通知
- 成功訊息（綠色）
- 錯誤訊息（紅色）
- 自動消失

### Loading 狀態
- 頁面載入中
- 按鈕載入中
- 區塊載入中

---

## 響應式設計

| 裝置 | 佈局 |
|------|------|
| Desktop | 完整側邊欄 + 內容區 |
| Tablet | 可收合側邊欄 |
| Mobile | 底部導航 + 全寬內容 |

---

## 設計風格建議

- 簡潔、專業
- 主色調：慈濟藍或中性色
- 字體：易讀的中文字體
- 圖示：使用 icon library（如 Lucide、Heroicons）
